# EES Construction Landing (React + TypeScript)

A premium multi-lingual landing page concept for **EES Construction** focused on craftsmanship, completed projects, and a modern Cyprus-forward visual identity.

## Languages included
- English (`en`)
- Turkish (`tr`)
- Russian (`ru`)
- Arabic (`ar`, RTL)
- Farsi/Persian (`fa`, RTL)

## Design system (v1)

### Font stack
- **Primary UI**: `Inter`
- **Display / headings**: `Manrope`
- **Arabic & Persian support**: `Noto Naskh Arabic`

### Color scheme
- Background: `#080B11`
- Surface: `#101826`
- Elevated surface: `#152033`
- Accent gold: `#D4AF62`
- Primary text: `#EBF0FF`
- Secondary text: `#A6B0C7`
- Borders/lines: `#2F3C56`

### Motion direction
- Scroll reveal animation using `animation-timeline: view()` for immersive section transitions.
- Sticky glassmorphism navigation for high-end visual feel.
- Layered gradient surfaces to emulate architectural depth.

## Suggested image direction for your final photos
Use high-resolution (at least 2400px wide) photos in these categories:
1. Structural frame + concrete precision shots
2. Exterior facade with dramatic daylight and sky reflections
3. Interior craftsmanship (staircases, stone, wood, lighting)
4. Finished villas / apartments with lifestyle context
5. Team-at-work moments emphasizing professionalism and safety
